=== GESTOR DE EVALUACIONES UNIVERSITARIAS ===

1. Ejecuta "GestorNotas_Setup_v1.0.exe" y sigue los pasos


 Para sincronizar con Drive

1. Copia el archivo "credentials.json" a la carpeta donde se instaló:
   Normalmente: C:\Program Files\GestorNotas\

2. Abre la aplicación desde el menú Inicio o escritorio

3. La primera vez que sincronices:
   - Se abrirá tu navegador
   - Inicia sesión con tu cuenta de Google
   - Acepta los permisos
   - Listo, ya puedes usar la app
